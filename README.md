# 移动端响应式应用

基于 React 18 + Antd Mobile 构建的响应式移动端应用，支持多端适配。

## 技术栈

- **React 18** - 前端框架
- **Vite** - 构建工具
- **Antd Mobile** - 移动端 UI 组件库
- **React Router** - 路由管理
- **CSS3 媒体查询** - 响应式布局

## 功能特性

- 📱 移动端优先的响应式设计
- 🎨 现代化 UI 界面
- 🧭 底部导航栏
- 📄 多页面路由
- 🔧 完整的项目结构

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

```
src/
├── components/     # 公共组件
├── pages/         # 页面组件
├── styles/        # 样式文件
├── hooks/         # 自定义钩子
└── utils/         # 工具函数
```
